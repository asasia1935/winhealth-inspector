# WinHealth Inspector

Windows의 시스템 상태를 투명하게 진단하고, 사용자가 선택한 안전한 관리 작업을 수행하는 오픈소스 WPF 프로그램입니다.

> 현재 프로젝트는 제품 정의 및 아키텍처 설계 단계입니다. 아직 실행 가능한 버전은 제공되지 않습니다.

## 프로젝트 목표

기존 PC 관리 프로그램은 종종 문제 개수를 과장하거나, 정확한 근거 없이 시스템 설정 변경을 권장합니다.

WinHealth Inspector는 다음 원칙을 따릅니다.

- 모든 진단 결과에 실제 측정값과 판정 근거를 표시합니다.
- 시스템 상태와 검사 실패를 구분합니다.
- 사용자의 명시적인 동의 없이 시스템을 변경하지 않습니다.
- 관리 작업을 실행하기 전에 대상과 예상 결과를 미리 보여줍니다.
- 기본적으로 일반 사용자 권한으로 실행합니다.
- 사용자 데이터와 진단 정보를 외부 서버로 전송하지 않습니다.
- 레지스트리 일괄 청소나 근거 없는 성능 향상을 제공하지 않습니다.
- 저장 공간 확보와 실제 성능 향상을 구분하여 설명합니다.

## MVP 범위

### 진단

- 저장 공간 상태
- 최근 프로그램 및 시스템 오류
- Windows 보안 상태
- Windows 업데이트 및 재부팅 상태
- 시작 프로그램 상태
- 네트워크 기본 상태

### 관리 작업

- 오래된 사용자 임시 파일 선택 정리
- 휴지통 용량 확인 및 명시적 비우기
- DNS 캐시 초기화
- 관리 작업 이후 관련 진단 재실행 및 전후 비교

## 기술 방향

- C#
- WPF
- .NET
- MVVM
- Windows API
- 단위 테스트

정확한 .NET 버전, 테스트 프레임워크, 패키징 방식은 구현 시작 전에 확정합니다.
