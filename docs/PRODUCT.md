# Product Definition

> 이 문서는 초기 제품 정의 초안입니다. 구현 과정에서 검증된 내용에 따라 수정할 수 있습니다.

## 한 문장 정의

WinHealth Inspector는 Windows의 시스템·보안·저장 공간·오류·네트워크 상태를 진단하고, 사용자가 선택한 안전한 관리 작업을 수행한 뒤 작업 전후 결과를 비교해 주는 오픈소스 WPF 프로그램이다.

## 해결하려는 문제

사용자는 PC 관리 프로그램이 표시하는 문제에 대해 다음 내용을 알기 어렵다.

- 실제로 문제가 있는가?
- 무엇을 검사했는가?
- 어떤 기준으로 문제라고 판단했는가?
- 제안된 변경이 안전한가?
- 변경 후 실제로 상태가 개선되었는가?
- 프로그램이 사용자 정보를 외부로 전송하는가?

WinHealth Inspector는 모든 진단과 조치에 근거를 제공하여 사용자가 스스로 판단할 수 있도록 돕는다.

## 핵심 사용자

- 자신의 Windows PC 상태를 확인하고 싶은 사용자
- 오랫동안 PC 상태를 점검하지 않아 현재 상태를 한 번에 확인하고 싶은 사용자
- 반복적인 프로그램 오류나 네트워크 문제의 원인을 확인하고 싶은 사용자
- 가족이나 지인의 PC를 점검하는 사용자
- 투명한 오픈소스 Windows 관리 도구를 원하는 사용자

## 핵심 사용자 흐름

```text
전체 진단 실행
→ 진단 결과와 판정 근거 확인
→ 사용 가능한 관리 작업 확인
→ 변경 대상과 예상 결과 미리보기
→ 사용자가 작업을 명시적으로 선택
→ 작업 실행
→ 관련 진단 재실행
→ 작업 전후 결과 비교
```

## 핵심 원칙

### 1. 투명성

모든 진단 결과에는 가능한 범위에서 다음 항목을 포함한다.

- 실제 측정값
- 진단 상태
- 판정 기준
- 예상 영향
- 권장 조치
- 검사 시간 및 실패 이유

### 2. 사용자 선택

시스템을 변경하는 작업은 사용자가 명시적으로 선택해야 한다.

- 자동 실행하지 않는다.
- 변경 전 대상과 예상 결과를 보여준다.
- 되돌릴 수 없는 작업은 별도로 경고한다.
- 실패와 일부 성공을 숨기지 않는다.

### 3. 최소 권한

일반적인 진단과 사용자 영역의 작업은 관리자 권한 없이 수행한다.

MVP에서는 관리자 권한 상승을 직접 수행하는 기능을 제외한다.
관리자 권한이 필요한 항목은 실행 불가 또는 권한 필요 상태로 안내한다.

향후 관리자 작업을 지원할 경우 애플리케이션 전체를 관리자 권한으로 실행하기보다 제한된 별도 프로세스를 검토한다.

### 4. 로컬 처리

- MVP에서는 로그인과 별도 애플리케이션 서버를 제공하지 않는다.
- 진단 정보와 시스템 정보는 기본적으로 사용자의 PC 안에서 처리한다.
- 사용자의 명시적인 동의 없이 시스템 정보나 진단 결과를 외부로 전송하지 않는다.
- 텔레메트리를 기본 제공하지 않는다.
- 보고서 내보내기는 사용자의 명시적인 요청으로만 수행한다.
- 향후 배포 및 업데이트를 위해 GitHub Releases 등의 외부 배포 인프라를 사용할 수 있다.

### 5. 공포 마케팅 금지

- 문제 개수를 부풀리지 않는다.
- 검사하지 못한 상태를 위험으로 표시하지 않는다.
- 낮은 위험의 잔여 항목을 심각한 시스템 오류처럼 표현하지 않는다.
- 근거 없이 속도 향상이나 보안 개선을 보장하지 않는다.

### 6. 효과의 정직한 표현

저장 공간 확보와 성능 향상을 구분한다.

예를 들어 임시 파일을 삭제하면 저장 공간은 확보할 수 있지만, 일반적인 환경에서 컴퓨터 성능이 눈에 띄게 향상된다고 단정하지 않는다.

### 7. 진단과 조치의 분리

진단 결과가 존재한다고 해서 관리 작업이 반드시 필요한 것은 아니다.

- 진단은 현재 상태를 설명한다.
- 조치는 사용자가 선택할 수 있는 별도 기능이다.
- 조치가 없는 진단 결과도 허용한다.
- 조치 후에는 가능한 경우 관련 진단을 다시 실행한다.

## 비목표

- 백신 또는 악성코드 탐지 엔진 개발
- 레지스트리 일괄 청소
- Windows 서비스 무차별 비활성화
- 네트워크 레지스트리 자동 튜닝
- DNS 서버 강제 변경
- 메모리 강제 정리
- 사용자 파일 자동 삭제
- 브라우저 쿠키 및 로그인 정보 일괄 삭제
- 근거 없는 원클릭 성능 향상
- 기업용 중앙 관리 솔루션
- 실시간 하드웨어 모니터링 도구
- MVP 단계의 사용자 계정 및 클라우드 동기화
- 진단 데이터를 저장하거나 분석하기 위한 중앙 서버
